# BDR Helper Pro
### Prospecting Tool V1 Beta

Ferramenta de prospecção enterprise para times de BDR e AE.

## Módulos

- 🔍 **Busca** — Account mapping com dados reais
- 📁 **Contas** — Repositório persistente com status de prospecção
- 📬 **Sequências** — Cadências de 6 toques por stakeholder
- 📊 **Pipeline** — Kanban visual por estágio
- 📈 **Insights** — Dashboard de performance

## Setup

```bash
npm install
npm run dev
```

## Stack

- React 18 + Vite
- Persistência via window.storage (artifacts) ou localStorage
